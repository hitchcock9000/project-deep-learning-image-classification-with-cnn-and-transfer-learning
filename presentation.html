Project Overview

Project Description

Building and comparing deep learning models for image classification using the CIFAR-10 dataset.

Sample images from the dataset (32x32 pixels)

Objectives

  • Data Exploration: Understand & prepare CIFAR-10.
  • Custom CNN: Design & train from scratch.
  • Transfer Learning: Use MobileNetV2.
  • Evaluation: Compare performance metrics.

  1. Data Loading: Loading and merging CIFAR-10 batches.

Major Obstacle & Results

Unexpected Results

Detailed Metrics

Custom CNN outperforms across ALL metrics!

Learnings

  • Pre-trained models aren't magic bullets.
  • Input Shape Mismatch MobileNetV2 expects 224×224 images, but CIFAR-10 is only 32×32. This size difference destroys spatial features! : 32x32 vs 224x224.
  • Custom models can beat generic ones.

Conclusion

Key Insights

  • Custom CNNs are strong for specific tasks.
  • Transfer Learning requires domain adaptation.
  • Metrics beyond accuracy are vital.

Future Work
